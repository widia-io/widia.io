# KEFI — Site institucional + painel admin

Projeto fechado com **Tamyres** (dona da KEFI), marca autoral de slow fashion fitness em Sorocaba.

## Status

- **Fechado:** R$ 1.000 · pagamento único · 100% na entrega
- **Prazo:** 7 a 10 dias úteis
- **Hospedagem:** 1 ano incluso (domínio `.com.br` + Vercel/servidor próprio)
- **Data do fechamento:** Abril 2026

## O que a Widia faz

A Widia implementa soluções de inteligência artificial sob medida para empresas. Não vende software pronto nem licença de ferramenta — constrói soluções customizadas para o contexto de cada cliente.

## Serviços principais

### 1. Agentes de WhatsApp / Atendimento Inteligente
- Triagem e qualificação automática de leads
- Respostas a perguntas frequentes com linguagem do cliente
- Encaminhamento para o responsável correto
- Agendamento automático de reuniões
- Escalonamento para atendimento humano quando necessário

### 2. Produção de Conteúdo com IA
- Calendários editoriais mensais
- Scripts para Reels e vídeos curtos
- Adaptação de conteúdo técnico para redes sociais
- Posts para Instagram, LinkedIn, blog
- Reaproveitamento de material existente (palestras, artigos, livros)

### 3. Automação de Documentos
- Templates inteligentes com preenchimento automático
- Geração de documentos a partir de dados de formulário
- Variações automáticas por tipo de caso
- Integração com fluxo de trabalho existente

### 4. Automações operacionais diversas
- Integração entre sistemas
- Dashboards e relatórios automatizados
- Fluxos de trabalho otimizados com IA
- Monitoramento e alertas inteligentes

## Formato de proposta (padrão Widia)

### Estrutura recomendada
1. **Resumo executivo** — O problema e a solução em 3 linhas
2. **Contexto do cliente** — O que entendemos sobre o negócio
3. **Diagnóstico** — Onde identificamos oportunidades
4. **Solução proposta** — O que vamos implementar (foco na prioridade #1)
5. **Roadmap** — Fases, prazos e entregáveis
6. **Investimento** — Valores claros (setup + mensalidade quando aplicável)
7. **Próximos passos** — O que acontece se o cliente aceitar

### Regras de estilo
- Sem jargão desnecessário
- Frases curtas
- Números e prazos concretos sempre
- Máximo 3-4 páginas
- Visual clean, marca Widia
